Imperative Query Web Help Page

Imperative Query Web is a web interface to the Imperative Query language, which is a simple imperative query language that transpiles to json on a form that adheres to the WebDP /evaluate endpoint spec. To learn more about the syntax, you can visit the IQ github and read the README file.

Compilation

To compile your IQ query, you send it to the "/compile" endpoint. Instead of saving json files, the server will send back a list of json objects: 

            [
                {
                    "name" : "your query name",
                    "query" : {
                        "budget" : "<*QUERY BUDGET*>",
                        "dataset" : "<*DATASET ID*>",
                        "query" : [ "... YOUR QUERY STEPS" ]

                    }
                }
            ]
where the value of the outer 
"query"
is your WebDP evaluate request.

Editor

A simple editor is available from the endpoint /editor

Summary

The server runs on port 3000. The endpoints that are supported are:

  1. /compiler (POST)
  2. /editor (GET)
  3. /help (GET)
